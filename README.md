# Days of the Past

A minimal web app that visualizes your entire life as a grid of dots — one dot per day (or week) across a 75-year lifespan.

## Features

- **Dot grid** — black dots for days lived, a pulsing blue dot for today, pale dots for days ahead
- **Days / Weeks toggle** — switch between 27,394 daily dots or 3,913 weekly dots
- **Life stats** — days lived, your age, and a progress bar showing life elapsed
- **Milestones** — interesting facts tied to each age you've crossed
- **Shareable card** — generates a 1200×675 PNG snapshot to save or share

## Usage

No build step. Just open `index.html` in a browser.

## Deploy

```bash
npm i -g vercel
vercel
```
